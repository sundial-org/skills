# Sundial for Skills

- `cli/`: Sundial CLI is a simple way to manage agent skills locally. Install with `npm i -g sundial-hub` then add skills from a Github url, the [Sundial registry](https://sundialscientific.com), or a local folder with `sun add`. See more in the [cli's README.md](cli/README.md)
- `skills/`: library of curated research skills that are also at   [sundialscientific.com](https://sundialscientific.com). Add them with `sun add <skill-name>`
